# 键盘快捷键

Chronicle 中的所有键盘快捷键，按适用场景分组。

Chronicle 以键盘操作为先：你无需触碰鼠标，就能打开会话、在时间轴上移动，并对一段对话进行提炼。快捷键按上下文注册，因此同一个按键在 Playback 与 Refine 中含义可能不同。下文中，`⌘` 是 macOS 的 Command 键——在 Windows 和 Linux（源码构建）上，它在各处都映射为 `Ctrl`，因为每个处理器监听的都是 `metaKey || ctrlKey`。

## 全局

在应用的任何位置都可使用。

| 快捷键 | 操作 |
| --- | --- |
| `⌘K` | 打开命令面板（跨会话的全局搜索） |

## 会话视图

在会话打开时始终生效。这些快捷键用于切换模式并驱动搜索过滤。

| 快捷键 | 操作 |
| --- | --- |
| `⌘F` | 聚焦会话内搜索框 |
| `⌘1` | Overview 模式 |
| `⌘2` | Playback 模式 |
| `⌘3` | Refine 模式 |
| `⌘4` | Replay 模式 |
| `⇧⌘U` | 同步（重新导入）当前会话 |
| `Esc` | 清除搜索过滤并取消搜索框聚焦 |

## Playback

在 Playback 模式下代码面板正在显示某个快照时可用。

| 快捷键 | 操作 |
| --- | --- |
| `D` | 切换所选文件的 diff 视图（在输入字段中打字时忽略） |

当焦点位于 `input` 或 `textarea` 中时，`D` 切换会被抑制，因此输入文件名过滤条件时绝不会翻转视图。

## 时间轴

在时间轴滑块获得焦点时生效（先点击它）。时间轴会将整个会话定位到某个时间点。

| 快捷键 | 操作 |
| --- | --- |
| `←` / `→` | 将播放头微调 1% |
| `Home` | 跳转到会话开头 |
| `End` | 跳转到会话结尾 |
| 点击 / 拖动 | 定位到指针位置 |

## Refine

在 Refine 模式下生效——你在此通过保留、删除、编辑和插入消息，将一段对话提炼为可分享的核心内容。

| 快捷键 | 操作 |
| --- | --- |
| `K` | 保留所选消息并前进到下一条 |
| `D` | 删除（划除）所选消息并前进到下一条 |
| `E` | 就地编辑所选消息 |
| `I` | 在所选消息之后插入一条备注 |
| `↓` / `j` | 向下移动选择 |
| `↑` | 向上移动选择（`⇧K` 也可向上移动） |
| `⌘Z` | 撤销 |
| `⇧⌘Z` | 重做 |
| `⌘S` | 导出提炼后的对话 |
| `Esc` | 取消当前的就地编辑 |

> **注意：** `K` 单独按下时的作用是保留一条消息，因此它在这里并不是 vim 的“向上”键——请使用 `↑`（或 `⇧K`）向上移动，使用 `j` / `↓` 向下移动。

## 命令面板

在 `⌘K` 面板打开时生效。

| 快捷键 | 操作 |
| --- | --- |
| `↑` / `↓` | 在结果间移动 |
| `Enter` | 打开高亮的结果 |
| `Esc` | 关闭面板 |
