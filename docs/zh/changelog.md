# 更新日志

Chronicle 的重要变更。完整历史与下载：
https://github.com/chizhangucb/chronicle/releases

## v0.1.10 — 2026-07-12

一个更精准的会话指标，通过自动更新交付：

- **“Agent Active”（由“Active Duration”更名而来）** 现在能正确地衡量智能体的工作时长。它仍会排除
  你每次提问前的那段停顿，但不再把后台任务完成、应用内点击或中断标记算作“你在思考”——这些在日志里
  都带有 `user` 角色，所以一次后台构建结束会被错误地计入你的空闲时间。在一个真实会话里，这把数字从
  59 分钟跨度中的 33 分钟修正到了 43 分钟。卡片上的 ⓘ 提示会解释这一区别。

## v0.1.9 — 2026-07-12

主页与会话指标的改进，通过自动更新交付：

- **多选删除项目** —— 主页新增的 **Select** 模式会把项目卡片变成复选框，让你一次性从 Chronicle
  移除多个项目（全选 / 清除 + 内联确认）。你的源日志和文件夹绝不会被改动。
- **常驻工具栏** —— **Search**（⌘K）和 **+ Import Sessions** 现在在每个视图中都可用，不再仅限主页。
- **更真实的活跃时长** —— 现在会完整计入助手的思考与工具执行时间，只排除你每次提问前的那段停顿
  （你阅读/输入/离开的时间）。旧版本会丢弃任何超过 5 分钟的间隔，从而低估了长时间的构建和思考。
- **缓存写入成本按 TTL 拆分** —— Cost & Usage 面板会把缓存写入的令牌和美元拆分为 **5 分钟** 和
  **1 小时** 两档，对应各自的计费方式。
- **可读的提示** —— ⓘ 信息气泡向下展开且更宽，因此完整的说明始终可见。

## v0.1.8 — 2026-07-09

下载站点 + 开发者文档。**没有应用改动**——桌面应用与 v0.1.7 完全相同，所以没有可安装的更新
（就像 v0.1.4 的文档发布一样）。

- **[getchronicle.dev](https://getchronicle.dev)** —— 一个简洁的下载页面，自动识别你的操作系统，
  并提供一个已签名并公证的下载，取代了原始的 GitHub Releases 列表。它实时读取最新发布，因此新版本
  （以及未来的 Windows/Linux 构建）会自动出现。支持浅色 + 深色、移动端自适应，并配有一张真实的
  应用内 Playback 截图。作为一个新的 `website/` 可部署项构建，部署在 Vercel 上。
- **开发者文档** —— 一套分层的 `docs/`（指南、架构、参考），涵盖安装、时间旅行、MCP/Skills、
  解析器、打包、安全等等。

## v0.1.7 — 2026-07-09

会话 UX 打磨，通过自动更新交付：

- **⇧⌘U** —— 用键盘同步单个会话。
- **活跃时长 (Active Duration)** —— 会话概览现在会在挂钟时长旁显示真实的工作时间（超过 5 分钟的
  空闲间隔被排除），并附有一个 ⓘ 说明。
- **提炼 → 按类型删除** —— 一键保留或丢弃整类消息（User / Assistant / Tool Call / …）。
- **一致的聊天标签** —— Playback 和 Refine 现在使用相同的措辞。
- **回放** —— 更整洁的沙箱工具栏。
- **反馈** —— 可选的发件人邮箱，被设为 `Reply-To`，让回复能送达你。
- **修复** —— 切换语言不再跳回主页。

## v0.1.6 — 2026-07-09

第一个**已签名并公证**的版本。

- **一键自动更新** —— 一个应用内 “Relaunch to apply” 提示条会安装更新并干净地重启
  （在已公证的 Developer ID 构建上运行 electron-updater）。
- 反馈迁移到 **getchronicle.dev**。
- 不再有 macOS 隔离标记。（0.1.5 是未签名的——手动升级到 0.1.6 一次，之后每次更新都是自动的。）

## v0.1.5 — 2026-07-08

成本与用量、全局搜索、会话标题、日语。

- **成本与用量 (Cost & Usage)** 面板 —— 本地按模型的 token 总量 + 金额明细（不联网）。
- **全局搜索**面板（⌘K），覆盖所有会话内容，带范围和筛选。
- 会话概览上的 **Skill** 与 **MCP** 分发甜甜圈图。
- 读取 Claude Code 的 `/rename` 标题；内联重命名；按会话的 Sync Update。
- 日语 (日本語) UI；项目切换下拉；“Today” 时间筛选。
